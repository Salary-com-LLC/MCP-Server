What is MCP (Model Context Protocol)

Model Context Protocol (MCP) is a standardized interface that allows AI systems to securely and consistently access external data sources in real time. MCP enables AI applications—such as chatbots, copilots, and agents—to retrieve trusted, structured information and return it directly within the AI conversation context.
In simple terms, MCP allows AI to “plug into” authoritative data and use it intelligently during a user interaction.

•	Ensure consistent, explainable, and auditable outputs

How the Job Match MCP Works

The Salary.com Job Match MCP uses AI to analyze job context provided during a conversation and maps it to Salary.com’s trusted job and compensation database.

Input (Provided by AI Chat Context)

The MCP accepts structured or unstructured job context, including:

•	Job Title

•	Job Description

•	Location (optional)

•	Country (optional)

•	Company Name (optional)

The AI can pass any combination of these fields, making the MCP flexible for real-world conversations.
________________________________________
MCP Output (Returned to AI)

The Job Match MCP returns standardized, high-value compensation insights:

•	Salary25

25th percentile salary (lower market range)

•	Salary50

50th percentile salary (market median)

•	Salary75

75th percentile salary (upper market range)

•	MatchRating

A confidence score indicating how closely the provided job context matches a known Salary.com benchmark role

These results are optimized for direct use in AI responses, dashboards, or downstream decision-making systems.
